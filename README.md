### Features
- Model importer (PMX, X)
- Keyframe based Animation (VMD)
- Rigid body physics (Bullet)
- Inverse IK (CCD)
- Shadow Map (LiSPSM)
- Skinning (LB, DQB)
- AA (MSAA, SMAA, FXAA)
- HDR
- Bloom

### Effects
- Auto Luminous4
- Diffusion7
- 多光源エフェクト
- Mirror
- Full

### DIRECTX 11.1
- Ported from [MiniEngine](https://github.com/Microsoft/DirectX-Graphics-Samples/) DIRECTX 12
